Works with the CASH Music platform to allow element inserts via WP short-code.

== Description ==

This plugin links an existing install of the CASH Music platform to your WordPress
install and allows you to easily insert elements via short-code. 

== Installation ==

Download the package, extract it, and upload the contents to your 
`wp-content/plugins/` directory. Activate it from the plugins menu. You'll need
to include the full path to your local cashmusic.php file as well as the email 
address of your main CASH admin account. You can find all the info you need by 
logging into your CASH installation and looking under "System Settings" for the 
full installation path.
